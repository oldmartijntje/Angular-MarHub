Casus / inhoud van de opdracht
Bouw een MPA/SPA volgens het Trello board: Trello MPA - Jukebox / Playlist (https://trello.com/b/WvAtIu4G/mpa-spa-jukebox-playlist)

Gebruik hiervoor een taal en framework naar keuze. Het advies gaat uit naar Laravel in de taal PHP.

Zorg ervoor dat je de uitwerking heb gecommit en gepushed naar een github repository. De link naar de github repository lever je in voor deze opdracht. ​

 

Functionele eisen: 

Zie de functionele eisen op het trellobord
Technische eisen (zie ook de technische eisen op het trellobord). 

Je maakt gebruik van een framework in een taal naar keuze (dit framework heb je netjes overlegt met je docent).
Maak gebruik van een codepatroon met gescheiden lagen (MVC/MVVM of iets dergelijks).
Je maakt gebruik van een Templating engine om je pagina te genereren.
Je maakt gebruik van een ORM om met een database te interacten. 
Het is de bedoeling dat je NIET handmatig database tabellen aan gaat maken, maar de hiervoor iets gebruikt als “migrations” hiervoor gebruikt. Elke tabel in je database moet terug te leiden zijn naar een migration file.
Gebruik seeders om testdata in te laden.
Onderzoek de backlog op het Trello board en inventariseer de benodigde tabellen, hun kolommen en hun onderlinge relaties. 

De database bevat waarschijnlijk tabellen als:

songs
genres
saved_lists
saved_list_songs
users
